# 🎮 Tic-Tac-Toe (WinForms, C#)

A modern **Tic-Tac-Toe (X/O)** game built using **C# Windows Forms** with an unbeatable AI powered by the **Minimax algorithm** 🤖.

---

## ✨ Features

* 🪟 Windows Forms (GUI)
* 🤖 Smart AI (Minimax algorithm)
* 🎯 Human vs Computer gameplay
* 🔴 X = Red, 🔵 O = Blue (colored UI)
* 🏆 Win / Lose / Draw detection
* 🔄 Auto reset after game ends
* 💡 Clean and simple interface

---

## 🧠 AI Logic

The game uses the **Minimax algorithm**, which ensures:

* The AI always plays optimally
* It never loses
* It evaluates all possible moves recursively

## 🚀 Getting Started

### 🔧 Requirements

* Visual Studio 2019 or later
* .NET Framework / .NET (WinForms)

### ▶️ Run the project

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repo-name.git
```

2. Open the solution file (`.sln`) in Visual Studio

3. Press **F5** to run

---

## 🎮 How to Play

* Click any empty cell to place your move
* You play as **X or O**
* The computer will respond instantly
* First to align 3 symbols wins
